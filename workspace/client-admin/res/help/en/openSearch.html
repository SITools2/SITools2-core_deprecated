Save an openSearch index

Follow the steps below to save an openSearch index. 

Save an openSearch index:

  1. Enter a name, a description and an image.
  2. Select the fields to index.
  3. Select the return type for each field. Those return types are RSS types.GuidField must be the primary key of the dataset in order to see the details of the data in the user part.
  4. Select the field searched by default.
  5. Select the unique key field of the index. This field has to be unique for all the data indexed, use the dataset's primary key can be a good idea.
  6. Select the fields used for keyword autosuggestion. Those fields will be used for autosuggestion, in order to help the user searching its data.
  7. Select the solrType used for the indexing. The default type is Text. The solrType of  the unique key must be string.
  8. Click Save

Activate an openSearch index

Click on Activate index to activate the index. The data will be indexed and you will be able to query the index.

    Note:
  1. Make sure to save the index prior to the activation, otherwise your changes won't be used.
  2. Be aware that the activation can take some time depending on the number of data.

Disable an openSearch index

Click on Disable index to disable the index. 

Refresh an openSearch index

Click on Refresh index to refresh the index.

    Note:
  1. Be aware that the refreshing can take some time depending on the number of data.

Delete an openSearch index

Click on Delete index to delete the index.
