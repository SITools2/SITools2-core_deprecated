This panel allows the administrator to add new form components for the query form.

Create a form component

  1. Click on Create
  2. Fill in the type, selection type, default height, default width, javascript object and image
  3. Click onOk

Edit a form component

  1. Select a form component  in the list
  2. Click on Edit.
  3. Fill in the type, selection type, default height, default width, javascript object and image
  4. Click on Ok

Delete a form component

  1. Select a form component  in the list
  2. Click on Delete
