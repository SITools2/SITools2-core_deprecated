Create a Feed

  1. Select the project in the ComboBox.
  2. Click on Create.

    3. Feed details panel

    1. Enter the title, the description, the link, the author name, the author email and the feed type.

    Feed items panel

    1. Add an item to the feed
      • Click on Add 
      • Enter the title, the description, the link, the author name, the author email and both updated and published dates
      • Click on Ok
    1. Edit an item of the feed
      • Select an item from the list
      • Click on Edit 
      • Edit item's details
    1. Delete an item of the feed
      • Select  an item in the list
      • Click on delete
  1. Click on Ok
    Note:    
  1. The items of a feed with an Opensearch source cannot be changed

Edit a Feed

  1. Select the project in the ComboBox.
  2. Select a feed from the list
  3. Click on Edit 
  4. Edit feed's details

Delete a Feed

  1. Select the project in the ComboBox.
  2. Select a feed from the list
  3. Click on Delete
