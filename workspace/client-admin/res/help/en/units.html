Create a dimension

  1. Click on Add.
  2. Choose a Dimension helper class name from the list.
  3. Fill in the form with a name and a description in the panel Dimension details.
  4. In the panel Select converters you can chose converters class for units compatibility.
  5. In the panel Select units you can define units for the dimension by a click on Create
  6. Click on Delete for removing a unit definition from the dimension.

Edit a dimension

  1. Select the dimension.
  2. Click on Edit.
  3. Fill in the form with a name and a description in the panel Dimension details.
  4. In the panel Select converters you can chose converters class for units compatibility.
  5. In the panel Select units you can define units for the dimension by a click on Create
  6. Click on Delete to remove a unit definition from the dimension.

Delete a dimension

  1. Select the dimension.
  2. Click on Delete.
