A collection is a set of datasets. You need to create a collection when you want to create multiple Datasets forms.

Create a Collection

  1. Click on Create

    2. Collection information panel

        Enter:

    1. the name: Name of the collection.
    2. the description: A short description.

    Select datasets panel

        This panel is used to select the datasets to include in the collection

    Add datasets:

    1. Click on Add
    2. Select some datasets in the list 
    3. Click on Ok to validate

    Remove a dataset:

    1. Select a dataset in the list 
    2. Click on Remove to remove it from the list
  2. Click on Ok

Edit a Collection

You can edit collection information.

To edit collection information:

  1. Select the collection.
  2. Click on Edit and edit the collection information.
  3. Click on OK to finish.

Remove a collection

  1. Select the collection.
  2. Click on Delete and then Yes on the prompt dialogue for confirmation.
