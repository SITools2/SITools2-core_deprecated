Create a form

  1. Select the project in the ComboBox.
  2. Click on Create.
  3. Fill in the form with a name and a description in the Form Information panel.
  4. Panel properties allows you to create, modify and delete properties to request Datasets.
  5. Panel Components allows you to create, modify and delete components.
  6. Panel Disposition allows you to choose the position of each of your components

Edit a form

  1. Select the form in the list.
  2. Click on Edit.
  3. Fill in the form with a name and a description in the Form Information panel.
  4. Panel Components allows you to create, modify and delete components.

Remove a form

  1. Select the form in the list.
  2. Click on Delete.

Create a form component

  1. In the panel Component, click on Create to create a new component 
  2. You have to choose the component you want to add. 

Creating or editing an HL (horizontal Line).

You can specify a specific CSS, height and width.

Creating or editing an IMAGE component

You can specify a specific CSS, height and width. You have to choose an image with the wizard, or specify a valid URL.

Creating or editing a LABEL component

You can specify a specific CSS, height and width and the label you want to add.

Creating or editing a TEXTFIELD component

You can specify a specific CSS, height and width, a default value.

Check AutoComplete if you want to add auto Complete services for this text field.

 Creating or editing a DATE_BETWEEN, NUMBER_BETWEEN, ONE_OR_BETWEEN component

You can specify a specific CSS, height and width, a default value.
