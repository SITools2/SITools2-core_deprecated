The purpose of the "order" component is to follow up the different downloads done by the order SVA. In addition, this order component allows for processing off line data. The administrator can look at different off line orders asked by the user. The "Enable" button means that the administrator is packaging data asked by user. The "Finish" button means the package is being delivered to the user.

Show details of an order

  1. Select the order.
  2. Click on Details.

Enable

  1. Select the order you want to activate.
  2. Click on Enable to activate the order.
  3. Enter a description and a message for this order.
  4. Click on OK to validate the activation, or Cancel to cancel modifications.

Finish

  1. Select the prepared order.
  2. Click on OK to validate your choice, or Cancel to cancel modifications.
