For each new user, a user storage is automatically created. A user storage is used to store window preferences of application module. In addition, it is also a free space on the server for the user where some asynchronous service can store results.

Create a user storage

  1. Click on Create. Enter the user storage name and description.
  2. Click on OK to finish.

Edit a user storage

  1. Select the user storage you want to edit.
  2. Click on Edit and edit the user storage info.
  3. Click on OK to finish.

Delete a user storage

  1. Select the user storage you want to delete.
  2. Click on Delete and then Yes on the prompt dialogue for confirmation.

Enable a user storage

  1. Select the user storage you want to activate.
  2. Click on Enable.

Disable a user storage

  1. Select the user storage you want to disabling.
  2. Click on Disable.

Clean a User Storage

  1. Select the user storage you want to clean.
  2. Click on Clean and then Yes on the prompt dialogue for confirmation.

Refresh a User Storage

  1. Select the user storage you want to refresh.
  2. Click on Refresh.

Notify

The administrator can send a message to a user when no free space is available
  1. Select the user storage you want to notify.
  2. Click on Notify.

Email limitation:

The message that is sent to the user is not customizable for this version.
