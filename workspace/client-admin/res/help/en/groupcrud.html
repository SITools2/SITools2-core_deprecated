A group is a collection of user accounts. It is useful to consider creating a group when the administrator needs to set up the same security rights to a group of users. The notion of group is not predefined in SITools2. The administrator can then create whatever groups he needs. In addition, the administrator can edit and delete groups.

Create a group

  1. Click on Create. Enter the group name and description. See the section below for more information.
  2. Click on OK to finish.

Naming conventions:

  • The group name is case sensitive and should be from 1 to 32 Unicode characters, excluding the following symbols:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • The first character cannot be a minus sign or a space, and the last character cannot be a space.
  • The group description is case sensitive and limited up to 128 Unicode characters.

Edit a Group

You can edit groups information.

  1. Select the group to edit.
  2. Click on Edit and edit the group informations.
  3. Click on OK to finish.

Delete a group

  1. Select the group to delete.
  2. Click on Delete then Yes on the prompt dialogue for confirmation.

Add/delete group members

You can add users to the group you created, or delete them.

To add group members to a group:

  1. Select the group and click on Users.
  2. Click on Add.
  3. Select users from the member list, and click on Add. (To make multiple selections, press and hold the "Shift" or "Ctrl" key and click on the desired users.)
  4. Click on OK.

To delete members from a group:

  1. Select the group and click on Users.
  2. Select users from the member list.
  3. Click on Add.
  4. Click on OK.
