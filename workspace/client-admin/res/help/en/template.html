Create a template for a concept

  1. Click on Create.
  2. Fill in the form with a name and a description in the panel Template of concept information.
  3. In the panel Select properties you can create, modify and delete properties.

Edit a template

  1. Select the template.
  2. Click on Edit.
  3. Fill in the form with a name and a description in the panel Template of concept Information.
  4. In the panel Select Properties you can create, modify and delete properties.

Delete a template

  1. Select the template.
  2. Click on Delete.
