Firewall

The built-in firewall can prevent unauthorized logins, and control which services can be accessed. In addition, you can choose to allow or deny access to certain network ports from specific IP addresses.

To create a firewall rule:

  1. Click the General, LAN 1, LAN 2 (dual LAN models only), LAN, PPPoE, or Wireless Network tab, depending on the type of your network connection.
  2. Click Create.
  3. Apply the rules to specified ports by choosing any of the following in the Ports section:
    • Choose All to apply the rules to all ports.
    • Choose Select from a list of built-in applications.
    • Choose Custom and click Custom, and then enter up to 15 ports separated with comma, or specify a port range.
    • You can also specify the protocols.
  4. Allow or deny access from specified source IP addresses by choosing any of the following in the Source IP section:
    • Choose All to allow or deny all IP addresses.
    • Enter a Single host IP address.
    • Enter a Subnet.

Now you can see the created firewall rule in the rule list.

To delete firewall rules:

  • Uncheck the checkbox next to all firewall rules.
  • Delete all firewall rules.

Choose Allow or Deny in the Action section to allow or deny the source IP address to access the specified ports.

    Note:
  1. You can create up to 100 rules for a network interface, and you can reorder the rules with drag-and-drop functionality.
  2. Rules at the top will have higher priorities.
  3. When you combine 2 network interfaces with link aggregation, firewall will apply the rules of the first network interface, and reserve the rules of the second network interface.
  4. If your system has 2 network ports connecting to the same subnet, the firewall rules might not work properly.*

____
* Applies to specific models only.
