Create a dictionary

  1. Click on Create.
  2. Choose a template from the list of templates
  3. Fill in the form with a name and a description in the panel Dictionary information.
  4. In the panel Select Concept you can create, modify and delete concepts.
 Note:    
  1. You can only change the template if there are no concept defined

Edit a dictionary

  1. Select the dictionary.
  2. Click on Edit.
  3. Fill in the form with a name and a description in the panel Dictionary Information.
  4. In the panel Select Concepts you can create, modify and delete concepts.
 Note:    
  1. You can only change the template if there are no concept defined

Delete a dictionary

  1. Select the dictionary.
  2. Click on Delete.
