Sitools allows you to deal with a default authorization. This authorization is used when there is no other authorization defined on a given application.

It can be used to specify that an administrator can access everything and that public user can only have GET authorizations.

Edit an authorization


  1. Select the default authorization
  2. Click on Authorizations.
  3. A window opens with authorizations of the roles.
    • Create a role: click on Create to add one or more roles. Select all roles you want to add then click on OK to validate.
      One or more lines appear in the table then you can select all authorizations you want for the role:
      • all : can do everything
      • get : can consult resources without modify them
      • post : can create resources
      • put : can modify resources
      • delete : can delete resources
      • head :
      • options :
    • Delete a role : select a role in the list, click on Delete then "YES".
