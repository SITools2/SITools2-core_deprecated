Create a dictionary

  1. Click Create.
  2. Fill in the form with a name and a description in the panel Dictionary information.
  3. In the panel Select Clues you can create, modify and delete clues.

Edit a dictionary

  1. Select the dictionary.
  2. Click Edit.
  3. Fill in the form with a name and a description in the panel Dictionary Information.
  4. In the panel Select Clues you can create, modify and delete clues.

Delete a dictionary

  1. Select the dictionary.
  2. Click Delete.
