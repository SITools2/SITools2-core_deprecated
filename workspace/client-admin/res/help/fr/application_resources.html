Sitools allows you to add application resources as plugins.

Add a new instance of application resources

  1. Select a application in the application list
  2. Click on the Add Button
  3. In new Window Select a plugin class in the list
  4. In the next panel “Field mapping”, sets the value of the required application parameters.

Edit an application resource

  1. Select an application in the application list
  2. Select an existing application resource
  3. Click on the Edit Button
  4. In new Window update the Field mapping
  5. Click on Ok to validate.

Delete an application resource

  1. Select an application in the application list
  2. Select a application resource in the list
  3. Click on Delete
