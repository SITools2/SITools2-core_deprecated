This panel allows the administrator to add new dataviews in SITools2. Once a dataview is defined in the system, the administrator can set it for each dataset. Then, the result on the client side are presented under the dataview

Create a Dataset View

  1. Click on Create
  2. Fill in the name, description, js object, file url, image, and priority
  3. Click on Ok

Edit a Dataset View

  1. Select a Dataset View  in the list
  2. Click on Edit.
  3. update the name, description, js object, file url, image, or priority
  4. Click on Ok

Delete a Dataset View

  1. Select a Dataset View  in the list
  2. Click on Delete
