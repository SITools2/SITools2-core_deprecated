Create a graph

  1. Select the project in the ComboBox.
  2. Right click on the root node to access menu.
  3. Click on Create node to insert a new node, or
  4. Click on Add dataset to put a dataset access to the node

Create a node/dataset

  1. Follow previous steps to add a node/dataset
  2. Enter node information in the dialog, or
  3. Choose a dataset in the list
  4. Click on OK to validate

Edit a node/dataset

  1. Select the node in the graph
  2. Right click on the node
  3. Click on Edit
  4. Modify information in the dialog, or
  5. Choose a new dataset in the list.
  6. Click on OK to validate

Remove a node/dataset

  1. Right click on the node/dataset
  2. Click on Delete
