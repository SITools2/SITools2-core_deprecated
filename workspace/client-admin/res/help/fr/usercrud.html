The “Users” panel shows the list of user accounts. The panel displays the login, first name, last name and email of each user. The administrator can create, delete a user account or edit his user information.

Create a user account

  1. Click on Create.
  2. Enter the first name
  3. Enter the last name
  4. Enter the email
  5. Enter the login
  6. Enter the password
  7. Enter the confirm password.
  8. Fill a set of properties if needed

Username limitation:

The login is case sensitive and should be from 1 to 64 Unicode characters, excluding the following symbols:
! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

The first character cannot be a minus sign or a space, and the last character cannot be a space.

The user's firstname can include up to 32 Unicode characters.

The user's lastname can include up to 32 Unicode characters.

The email's lastname can include up to 128 Unicode characters.

Password limitation:

The password is case sensitive and is limited up to 40 displayable characters, including letters, numbers, signs, and space.

Email limitation:

The message that is sent to the user is not customizable for this version.

Edit a user account

Find the user account by entering the last name in the search field, select a row and click on Edit. The edit user information panel is the same as the create user information panel at the exception that the administrator cannot change the login.

Delete a user account

Find the user account by entering the last name in the search field, select a row and click on Delete. The user account will be definitively removed from the archive’s users.
