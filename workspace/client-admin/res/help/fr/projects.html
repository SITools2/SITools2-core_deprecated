The project component is the function that allows the administrator to display the different datasets and services on the web client (create, edit, delete, enable and disable).The following sections explain how to handle the project.

Create a project

  1. Click on Create

    2. Project information panel

        Enter:

    1. the name: Name of the project that is displayed on the portal and in the URI for accessing to project desktop
    2. the description: Project description is displayed on the portal
    3. the URI attachment: URI where information related to the project is located
    4. the image: Image that is displayed on the portal
    5. the CSS: Reference to the CSS class from the main.css file
    6. is always visible: when it sets to true, the projects is always displayed on the portal even if the user is not allowed to access it (a padlock is displayed). Otherwise, the project is hidden on the portal when the user is not allowed to access it.

    Select datasets panel

        This panel is used to select the datasets that are displayed on the web client

    Add datasets:

    1. Click on Add
    2. Select some datasets in the list 
    3. Click on Ok to validate

    Remove a dataset:

    1. Select a dataset in the list 
    2. Click on Remove to remove it from the list
  2. Click on Ok

Naming conventions:

  • The project name is case sensitive and should be from 1 to 15 Unicode characters, excluding the following symbols :

    ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • The first character cannot be a minus sign or a space, and the last character cannot be a space.
  • The project description is case sensitive and limited up to 64 Unicode characters.

Edit a project

You can edit project information.

To edit project information:

  1. Select the project.
  2. Click on Edit and edit the project information.
  3. Click on OK to finish.

Remove a project

  1. Select the project.
  2. Click on Delete and then Yes on the prompt dialogue for confirmation.

Enable a project

  1. Select the project you want to enable.
  2. Click on Enable

Disable a project

  1. Select the project you want to disable.
  2. Click on Disable.
