Create a dataset

  1. Click on Create and fill in the form:
    • Panel "Dataset information": set the dataset's general settings. The field "Datasource" is mandatory.
    • Panel "Select tables": choose the tables linked to the dataset.
      Select the tables to be associated in the left column then click on ›.
      You can associate all tables by pressing ››.
      To dissociate, click on tables in the right column then ‹.
      To dissociate all, click on ‹‹.

      To define an alias to a table, double-click on the alias column then enter the alias.
    • Panel "Select fields": choose the fields linked to the dataset.
      Select fields to associate in the left column then click on ›.
      To associate all fields, press the button ››.
      To dissociate, click on fields in the right column then ‹.
      To dissociate all, click on ‹‹.
    • Panel "Fields setup" : set parameters for fields. You can add or delete fields as well.
      "Assign clue" permits you to associate a concept of a dictionary to a field.
    • Panel "Criteria (Where clause)" : select criteria via a wizard or a standard SQL command line. This wizard permits to define a join condition and/or a where clause.
      • Wizard :  You can add some Sql restriction to your Select with this wizard : 
        • Join Condition :  Specify a join Condition between two tables. With a right mouse click on, you can 
          • add a condition  : 
            • Specify a logic operator by double clicking on the first cell
            • Specify some open parenthesis by double clicking on the second cell
            • Define a left attribute by double click on the left attribute and choose in the list a left attribute
            • Define an operator (this should be always "=")
            • Define the right attribute (same way than the left attribute)
            • Specify some close parenthesis by double clicking on the last cell
          • delete a condition (you must have select a condition before)
          • delete parenthesis : all the parenthesis of the selected conditions will be deleted
        • Where Clause : Specify some values for the Where Clause : 
          • See add a condition. The only difference is that you have to enter a value for the right attribute. 
      • SQL : If you don't want to use the wizard, you can specify all the Where clause of this datasets. 
        SQL clause must be valid, and begin with the keyword WHERE
  1. Click on OK to save and close or CANCEL to cancel modifications.

Activate a dataset

  1. Select the dataset storage you want to enable.
  2. Click on Active.

Disabling a dataset

  1. Select the dataset storage you want to disable.
  2. Click on Disable.
