Sitools allows you to add applications as plugins.

Add a new instance of pluggable application

  1. Select a plugin class in the list
  2. In the next panel, input the label and the User attachement url (starting with “/”)
  3. In the next panel “Parameters mapping”, sets the value of the required application parameters.

Edit a pluggable application

  1. Select a pluggable application in the list, the application must be INACTIVE in order to edit it
  2. Follow the same steps as add a new instance of pluggable application

Delete a pluggable application

  1. Select a pluggable application in the list
  2. Click on Delete

Start/Stop a pluggable application

  1. Select an pluggable application in the list
  2. Click on Enable or Disable to attach or detach the application from the server
